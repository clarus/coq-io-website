Coq.io

Voice your Coq.

Coq.io is a library to write Coq programs with IOs and concurrency. You code, you certify and you compile to a runnable OCaml program with verified IOs.

Try

Enable the repo-stable OPAM repository and run:

opam install -j4 -v coq:io:hello-world
